Binary Classification Project

Predicting customer churn using machine learning classification algorithms

Scikit-learn Pandas Classification Python

Project Overview

This project demonstrates a binary classification approach to predict customer churn for a telecommunications company. Using various machine learning algorithms, we analyze customer behavior patterns to identify customers likely to discontinue their service.

Dataset

  • Size: 7,043 customers
  • Features: 21 features including demographic, account, and service information
  • Target: Binary (Churn: Yes/No)
  • Source: Telecommunications customer data

Methodology

  1. Data Exploration: Analyzed feature distributions and correlations
  2. Data Preprocessing: Handled missing values, encoded categorical variables
  3. Feature Engineering: Created new features and scaled numerical data
  4. Model Training: Trained multiple algorithms (Logistic Regression, Random Forest, SVM)
  5. Model Evaluation: Compared performance using accuracy, precision, recall, and F1-score

Key Results

Random Forest

Accuracy: 85.2%

F1-Score: 0.78

AUC: 0.89

Logistic Regression

Accuracy: 82.1%

F1-Score: 0.75

AUC: 0.86

SVM

Accuracy: 83.5%

F1-Score: 0.76

AUC: 0.87

Feature Importance

The Random Forest model identified the following as the most important features for predicting churn:

  1. Monthly charges (importance: 0.24)
  2. Total charges (importance: 0.18)
  3. Contract type (importance: 0.16)
  4. Tenure (importance: 0.14)
  5. Internet service type (importance: 0.12)

Business Impact

The model can help the telecommunications company:

  • Identify high-risk customers before they churn
  • Implement targeted retention strategies
  • Reduce customer acquisition costs
  • Improve customer lifetime value
